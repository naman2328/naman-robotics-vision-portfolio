Why Vision Systems Fail in Factories

Real production environments introduce glare, vibration, inconsistent part positioning, and timing constraints. Naive vision pipelines that work in controlled conditions often fail without deterministic logic and validation.

OCR-Based Vision Inspection (Jetson Orin Nano)

Built an offline OCR inspection setup using PaddleOCR to understand how text detection and recognition behave on real camera images. The project helped me learn ROI selection, confidence handling, and basic OK / NG decision logic.

What I learned: OCR limitations, lighting impact, ROI importance, and basic inspection logic.

Tools: PaddleOCR · OpenCV · Python · Jetson Orin Nano

Vision-Guided Pick & Place (ROS 2 + Dobot)

Implemented a basic vision-guided pick-and-place system using ROS 2. I separated vision processing and robot control logic to better understand ROS node communication and real robot behavior.

What I learned: ROS 2 basics, vision-to-motion flow, and real hardware debugging.

Tools: ROS 2 · OpenCV · Python · Dobot Magician

Failure-First Engineering Approach

  • Glare & reflections → ROI constraints and preprocessing
  • False positives → validation rules & fuzzy matching
  • Latency issues → region-only processing
  • Unstable inputs → deterministic state handling
